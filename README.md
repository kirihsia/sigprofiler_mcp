# mutational-signature-analysis

An MCP (Model Context Protocol) server that wraps the AlexandrovLab
SigProfiler suite — SigProfilerMatrixGenerator, SigProfilerAssignment,
SigProfilerExtractor, and sigProfilerPlotting — as tools an LLM client
(Claude Code, Claude Desktop, or any MCP client) can call directly.

See [TUTORIAL.md](TUTORIAL.md) for the full walkthrough: setup, what each
tool does, how to test it, and how to connect it to a client.
